Nouns

Nouns are words that represent a person, thing, place or animal.

Countable Nouns

Feature Description Examples
Countable Nouns can be counted individually. dog (singular), dogs (plural)
Quantifiers Use with numbers and words like "many," "few." four bananas, many books
Plural Forms Indicate more than one. dog/dogs, book/books

Uncountable Nouns

Feature Description Examples
Uncountable Nouns can not be counted individually. water, sand, milk
Quantifiers Use with expressions like "some," "much," "a little." some water, a lot of information
No Plural Forms No plural form. furniture, advice
